# Vila Play Teleprompter — Landing Page

Arquivos incluídos:
- index.html
- styles.css
- script.js
- privacidade.html
- termos.html

## Antes de publicar

Abra o arquivo `index.html` em um editor de texto e substitua:

`financasnoautomatico@gmail.com`

pelo seu e-mail verdadeiro.

Exemplo:
`contato@suaempresa.com.br`

## Como testar

Clique duas vezes no arquivo `index.html`.

## Como publicar gratuitamente

Você pode usar:
- GitHub Pages
- Netlify
- Cloudflare Pages
- Vercel

Basta enviar todos os arquivos desta pasta mantendo os nomes originais.
